API

简要说明

jit

JIT 装饰器 - 使用 Triton 编译器编译函数

autotune

用于自动调优一个经过 triton.jit 编译的函数的装饰器

heuristics

用于指定如何计算某些元参数值的装饰器

Config

一个表示自动调优器可以尝试的可能内核配置的对象
