内存/指针操作

api

简要说明

load

返回一个张量，其值从由指针定义的内存位置加载

store

将数据张量存储到由指针定义的内存位置

make_block_ptr

返回指向父张量中某个块的指针

advance

推进一个块指针

load_tensor_descriptor

从张量描述符加载数据块

make_tensor_descriptor

创建张量描述符对象
